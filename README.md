# Attack of the Pancakes and Waffles

![](./assets/toaster.png)

A Shmup I made in Godot to practice making my own art. The focus was mainly on creating sprite art and using a monochrome color palette.

## Controls

| Action    | Key          |
| --------- | ------------ |
| Move Up   | `Up Arrow`   |
| Move Down | `Down Arrow` |
| Shoot     | `Space`      |
| Charge    | `Hold Space` |
